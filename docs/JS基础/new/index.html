new操作符#

  • new操作符做了哪些事情

    • 创建了一个对象
    • 把构造函数的this指向当前的对象o 并且向调用构造函数并传入参数
    • 把自定义对象的隐式原型指向构造函数的显式原型
    • 判断构造函数的返回值是基础值 还是对象（是基础值返回实力对象，是对象就返回构造函数的返回值）
