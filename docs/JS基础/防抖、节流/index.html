防抖、节流

防抖#

  • 防抖是指在一定的时间内再次触发此事件,会清空上次的事件重新开始,如果制定的时间内没有再次触发,那么这个事件才会执行。

    函数防抖就是法师发技能的时候要读条，技能读条没完再按技能就会重新读条。

     <body>    <div id="div">测试节流</div>    <script>      const oDiv = document.getElementById("div");      //防抖函数      const debounce = function (fn, time, ...other) {        let timeId = null;        return function () {          clearTimeout(timeId);          timeId = setTimeout(() => {            fn.apply(this, other);          }, time);        };      };      const move = function (a, b) {        console.log(111, this, a, b);      };      oDiv.onclick = debounce(move, 2000, 123, 456);    </script>  </body>

节流#

  • 节流是指在一定的时间同一事件只会触发一次,只有超过了这个时间才会再次触发。节流会稀释函数的执行频率。

    <body>    <div id="jl">测试节流</div>    <script>      const oDiv = document.getElementById("jl");      // 节流函数：在规定的一段时间内，只会触发第一次      const throttle = function (fn, time, ...other) {        let startTime = 0;        return function () {          let nowTime = new Date();          if (nowTime - startTime < time) {            return;          }          fn.apply(this, other);          startTime = nowTime;        };      };      const move = function (a, b)        console.log(111, this, a, b);      };      oDiv.onclick = throttle(move, 2000, 123,456);    </script>  </body>
