  • Scheduler（调度器）： 排序优先级，让优先级高的任务先进行reconcile
  • Reconciler（协调器）： 找出哪些节点发生了改变，并打上不同的Flags（旧版本react叫Tag）
  • Renderer（渲染器）： 将Reconciler中打好标签的节点渲染到视图上
